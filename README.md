# Weather App

A responsive weather application that fetches real-time forecast data using the OpenWeatherMap API and visualizes temperature trends with interactive charts powered by Chart.js.

---

## Features
- Search weather by city name  
- Real-time temperature forecast  
- Line chart visualization of temperature trends  
- Built using modern JavaScript (ES6, Async/Await)  
- Error handling for invalid inputs and API responses  

---

## Tech Stack
- HTML  
- CSS  
- JavaScript (ES6)  
- Chart.js  
- OpenWeatherMap API
